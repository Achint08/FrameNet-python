Contents
==========

- framenetlib.py -- The framenet data bindings generated from
  the framenet XML schema.

- framenetutils.py -- Utility functions etc. for use with the
  framenetlib module.

- ./docs/ -- Documentation on framenetlib.

- ./schemas/ -- The XML schemas from which these modules
  (framenetlib.py etc) were generated.

- ./sample_data -- Sample XML instance documents that whose
  document type is the framenet XML schema.

- ./sample_code -- Sample Python code that uses the framenetlib.py
  module.  See ./sample_code/README.txt.
